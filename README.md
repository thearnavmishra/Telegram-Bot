# Telegram E-Commerce Bot

A **Python-based Telegram bot** that automates product browsing and order management through the Telegram Bot API.  
The bot is deployed on **AWS EC2 inside a secure VPC**, ensuring controlled network access and continuous uptime.

---

## 🚀 Features

- Automated user interaction via Telegram commands  
- Product browsing and order management workflow  
- Real-time request handling using Telegram Bot API  
- Secure deployment using **AWS VPC networking**  
- Cloud VM hosting for **24/7 availability**

---

## 🛠 Tech Stack

- Python  
- Telegram Bot API  
- AWS EC2  
- AWS VPC
